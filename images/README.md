# Image Organization

Each case study has its own folder with the following convention:

## Naming Convention
- `thumbnail.png` — Small image shown on the homepage (recommended: 800×600px)
- `hero.png` — Large hero image shown at the top of the case study overlay (recommended: 1200×800px)
- Additional images can be named descriptively (e.g., `dashboard.png`, `user-flow.png`)

## Folders
- `monks-flow/` — Monks.Flow enterprise AI platform
- `google/` — Google AI products portfolio
- `zoox/` — Zoox autonomous vehicle service
- `adidas/` — Adidas Locker Room marketplace

## Image Specs
- Format: PNG
- Thumbnails: ~800×600px (or similar 4:3 aspect ratio)
- Hero images: ~1200×800px (or similar 3:2 aspect ratio)
- Additional images: optimize for web (under 500KB each)
